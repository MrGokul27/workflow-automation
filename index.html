Frequently Asked Questions

How secure is my data in Stackly? expand_more

We employ enterprise-grade encryption both in transit (TLS 1.3) and at rest (AES-256). Our platform is SOC2 Type II compliant and we offer self-hosted runners for complete data sovereignty.

Can I write custom code for complex logic? expand_more

Yes. While we offer 500+ pre-built blocks, you can drop in custom JavaScript or Python snippets within any workflow for specialized data processing or proprietary calculations.

What happens if an automation fails? expand_more

Our intelligent retry logic attempts execution based on your custom backoff strategy. If a terminal failure occurs, you are alerted via Slack/PagerDuty, and can replay any step with the original payload.

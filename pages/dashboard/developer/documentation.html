Core API Reference

The NexusFlow API provides programmatic access to our distributed workflow engine. Connect your existing infrastructure to automate deployments, monitor health metrics, and trigger sophisticated CI/CD pipelines.

POST /v2/workflow/trigger
link

Trigger Workflow

Initiates a predefined workflow sequence based on an external event or manual call.

Parameters

workflow_id

string REQUIRED

The unique identifier for the targeted workflow blueprint.

payload

object OPTIONAL

Custom JSON metadata to be passed into the workflow context.

Response Codes

202 Accepted

The request was valid and the workflow has entered the queue.

429 Rate Limit

Request quota exceeded. See headers for retry timing. 

import nexusflow

client = nexusflow.Client(api_key="sk_live_...")

response = client.workflows.trigger(
    id="wf_882910",
    payload={
        "env": "production",
        "force": True
    }
)

# Output workflow status
print(response.status)
